Installing on MAC OS X

The Embedthis Pak install distribution for Mac OS X is published as a an installable pkg file.

Installing the PKG Image

  1. Double click on the PKG image.

Removing Pak

  1. Run the remove script in the install directory for Pak by typing:
sudo ./uninstall

you can run pak

pak

To get all the options for Pak read the man page via:

man pak
or get the command help via: 
pak --help
